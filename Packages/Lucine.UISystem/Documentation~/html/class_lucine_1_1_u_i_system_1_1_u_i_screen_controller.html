Detailed Description

UIScreen controller is the base class for all types of windows, panels, popup whatever u want It implements the IUIScreenController interface It is templated depending of the Parameters we want for the window Parameters are thing you can add to customize the parameter of the window

Template Parameters
TParametersThe type of parameter for this screen. There are base parameters class for Windows and panel but you can extend them to add your own parameters
