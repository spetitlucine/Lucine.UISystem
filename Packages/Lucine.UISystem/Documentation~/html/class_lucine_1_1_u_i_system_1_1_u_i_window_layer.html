Detailed Description

Layer for windows inherited of base UILayerController It implements interface for UIWindowControllers Unlike panels layer, the window controller should keep track of transitionning windows to disable interactions during transition A panel is not aim to be interactable but only display information We also keep track of current window. Right now it is not used maybe in the future
