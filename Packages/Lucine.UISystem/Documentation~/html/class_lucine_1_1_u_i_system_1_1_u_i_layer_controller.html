Detailed Description

The layer class. The layer is responsible of all child screens, we should pass from the layer to show or hide screens The class is inherited with panel layers and windows layers Windows layer may have some features that panels don't have It is not implemented yet but we can think about handling history of windows when closing one window to show the previously opened one Layers are templated with the type of screen they handles One of the main role of this class is to make association of a screen with its id and have corresponding functions
